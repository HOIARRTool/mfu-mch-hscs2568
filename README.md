# HSCS Web Service

Deploy on Render with the bundled default Excel files, or upload your own from the sidebar.

## Local run
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Render
This repo already includes `render.yaml`.

Default bundled files:
- `HSCS2568_interac.xlsx`
- `plotgraph_quadrant_infographic.xlsx`
